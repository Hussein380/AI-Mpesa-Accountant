# AI-Pesa Project

This repository contains the AI-Pesa project, which consists of a frontend landing page 

## Project Structure

```
/
├── frontend/    # Next.js frontend application
├── backend/     # Backend application (to be developed)
```

## Frontend

The frontend is a modern, responsive landing page for AI-Pesa, an AI-powered M-Pesa accountant that helps users track and manage their transactions via WhatsApp.

See the [frontend README](./frontend/README.md) for more details.

## Backend

The backend will be developed in the future.

## Development

To run the frontend development server:

```bash
# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install --legacy-peer-deps

# Start development server
npm run dev
```

## Deployment

### Frontend Deployment

The frontend can be deployed to Vercel. See the [frontend README](./frontend/README.md) for deployment instructions.
